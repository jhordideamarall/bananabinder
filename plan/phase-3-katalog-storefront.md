# Phase 3 — Katalog & Storefront

## Objective
Build halaman publik: homepage, product listing, product detail dengan varian.

## Tasks

- [ ] Homepage layout (hero, featured products, categories)
- [ ] Product listing page (grid, filter, search)
- [ ] Product detail page (images gallery, variant selector, add to cart)
- [ ] Category pages (binder A5, B5, refill, accessories)
- [ ] Responsive design (mobile first)
- [ ] Image optimization (Next.js Image + Supabase Storage)
- [ ] Loading states & skeleton UI
- [ ] API: `GET /api/products` (with pagination, filter)
- [ ] API: `GET /api/products/[slug]` (detail + variants + images)

## Output
- User bisa browse produk, lihat detail, pilih varian
- UI responsive dan fast (< 1.5s load)
