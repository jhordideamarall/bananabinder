# Phase 6 — Admin Dashboard

## Objective
Build admin panel untuk manage produk, orders, marketing, dan analytics.

## Tasks

### Dashboard
- [ ] Analytics overview (total sales, orders today, conversion rate)
- [ ] Low stock alerts
- [ ] Recent orders widget
- [ ] Revenue chart (daily/weekly/monthly)

### Product Management
- [ ] Product list (table, search, filter)
- [ ] Create/Edit product form (name, description, price, weight, images)
- [ ] Variant management (add/edit/delete variants per product)
- [ ] Image upload (Supabase Storage)
- [ ] Activate/deactivate product

### Order Management
- [ ] Order list (filter by status)
- [ ] Order detail view
- [ ] Update status (processing → shipped)
- [ ] Input tracking number (resi)
- [ ] Print shipping label

### Marketing Management
- [ ] Coupon CRUD (create, edit, deactivate)
- [ ] Flash Sale CRUD (create, assign products, set period)
- [ ] Abandoned cart list view

### CRM
- [ ] Customer list (name, phone, total orders, total spent)
- [ ] Customer detail (order history)

## Output
- Admin bisa manage seluruh operasional dari dashboard
- Protected route (role: admin only)
