# Artifacts - Work History

## Purpose
Folder ini berisi riwayat lengkap setiap pekerjaan yang dilakukan di project Bananasbindery.

## Format Artifact
Setiap file artifact harus berisi:
- **Tanggal**
- **Phase**
- **Task yang dikerjakan**
- **File yang diubah/dibuat** (path lengkap)
- **Apa yang diubah**
- **Alasan/Rasionale**

## Naming Convention
```
YYYY-MM-DD-<deskripsi-singkat>.md
```
